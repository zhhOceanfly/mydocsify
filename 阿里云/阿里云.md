## 登录阿里云服务器
- 登录阿里云控制台管理
- ssh远程登录
  - 使用阿里云的密钥对配置
  - 配置ssh密钥进行远程登录
    ```shell
      ssh-keygen -R ip
      ssh-keygen -t rsa #输入private.key
        # 使用ssh-copy-id
          ssh-copy-id -i "/c/Users/zhh/.ssh/public.pub" "root@ip"
        # 或者手动拷贝 public.pub 到远程服务器的authorized_keys
    ```
  - 测试ssh连通性
    ```shell
      ssh -i "C:\Users\zhh\.ssh\private.key" root@ip -p 22
      ssh -T ip 
    ```
